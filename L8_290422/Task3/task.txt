Написать функцию которая:
принимает массив значений ["a", "b", "c"]
возвращает новый массив из значений отсортированных наоборот

Пример:

var res = sort(["a", "b", "c"])

Результат: ["c", "b", "a"]

Примечание:
Для реализации сортировки разрешается пользоваться только циклом
Запрещается пользоваться любыми функциями сортировки.
