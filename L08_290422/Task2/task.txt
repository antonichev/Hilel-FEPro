Написать функцию которая:
принимает массив чисел [2, 4, 3, 1] и степень числа 2
возвращает новый массив из чисел возведенных в указанную степень

Пример:
var res = exponentiate([2, 4, 3, 1], 2)

Результат: [4, 16, 9, 1]

Примечание:
* Запрещается пользоваться готовыми решениями и продвинутыми функциями js
* В основном нужно полагаться на знания полученные на уроках
* Для возведение в степень можно использовать оператор ** 4**2 = 16
* Для добавление новых элементов в массив используйте arr.push() вместо +=
